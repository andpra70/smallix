Smallix in TinyEMU (browser) TTY via VGA o seriale emulata Download smallix.img View cfg
Questa pagina usa TinyEMU/JSLinux lato browser. Device emulati disponibili: IDE disk (smallix.img), VGA, PS/2 keyboard, timer base. Device host reali (NIC/USB passthrough) non sono esposti direttamente nel browser. La modalita' TTY seriale usa il terminale HTML integrato.
Prima esecuzione: lancia ./tools/setup-tinyemu-web.sh e servi la root di smallix/ con HTTP (es. python3 -m http.server 8000), poi apri /www/tinyemu.html.
